¿Qué es CSS?

Breve descripción de CSS

Cascading Style Sheets (Hojas de Estilo en Cascada) forman las siglas CSS que corresponde a un lenguaje que maneja el diseño y la presentación de las páginas web. Su función se realiza en conjunto con HTML que se encarga del contenido básico.
